Máquinas Virtuais

As Máquinas Virtuais (VMs) são ambientes de computação virtuais que simulam a execução de um sistema operacional em um hardware físico. Na Azure, as VMs são usadas para executar aplicativos, hospedar serviços e simular redes. Elas permitem escalabilidade, flexibilidade e segurança.

Redes Virtuais

As Redes Virtuais na Azure fornecem comunicação segura e privada entre recursos, como VMs e bancos de dados. Elas funcionam como uma rede local, onde é possível segmentar, isolar e controlar o tráfego de dados.

Servidor de Aplicações

O Servidor de Aplicações é uma plataforma gerenciada na Azure para hospedar e executar aplicativos web. Ele oferece suporte a diversas linguagens de programação e frameworks, e garante alta disponibilidade e escalabilidade automática.

Banco de Dados

Os Bancos de Dados na Azure, como o SQL Database e o CosmosDB, são soluções para armazenar e gerenciar dados estruturados e não estruturados. A nuvem permite escalabilidade automática, backups e alta disponibilidade.

TCC - Monitoramento de Redes

O projeto de Monitoramento de Redes usa o Prometheus e o Grafana para coletar e visualizar dados sobre o desempenho de uma rede. A parte lógica do sistema foca na configuração do Prometheus para monitorar métricas como largura de banda e latência, e o Grafana para criar dashboards visuais.
